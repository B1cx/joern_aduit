## Authentication (D2) — Guided Questions

1. How are tokens (JWT/Session) generated? Is the secret key hardcoded or properly managed?
2. Is token signature verification actually performed (not just decoded)?
3. Are tokens validated for expiration?
4. Is there a secure password hashing mechanism (bcrypt/scrypt/argon2)?
5. Are there rate limits on authentication endpoints?
6. Can authentication be bypassed via alternative code paths?
